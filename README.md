# vue-portfolio

Voici comment installer et lancer le projet vue-portfolio

## IDE recommandé

[VSCode](https://code.visualstudio.com/)

## Project Setup

```sh
npm create vue@latest 

```sh
npm install @emailjs/browser --save
```

```sh
npm install sass
```


### Pour lancer le projet en local et développement

```sh
npm run dev
```
